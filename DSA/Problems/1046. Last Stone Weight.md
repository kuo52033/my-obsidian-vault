```python
import heapq

class Solution:
    def lastStoneWeight(self, stones: List[int]) -> int:
        max_heap = [-stone for stone in stones]
        heapq.heapify(max_heap)
        result = 0

        while len(max_heap) > 1:
            stone1 = heapq.heappop(max_heap)
            stone2 = heapq.heappop(max_heap)

            if stone1 != stone2:
                remain = abs(stone1-stone2)
                heapq.heappush(max_heap, -remain)

        return -max_heap[0] if len(max_heap) > 0 else 0
```

 這題最重要的是如何每次取出兩個最大的石頭，以及把剩下的再塞進去，如果每次 sort，複雜度會很高，因此決定維護一個 max_heap，先把一開始的石頭都塞進去，然後每次 pop 出兩個最大的石頭進行撞擊，把剩下的塞回去，直到剩下小於一顆石頭。那要注意 python 預設是  min_heap，因此要先轉成負數，這樣每次取出來的最小，才會是最大的石頭。
 time: 每次操作是 O(log n)（heappop 和 heappush 需要重新排列 heap） 最多操作 n 次，因此是 O(nlogn)
 space: heap 最多存 n 個石頭， O(n)
