```python
class Solution:

    def lengthOfLongestSubstring(self, s: str) -> int:
        char_set = set()
        result = 0
        left = 0

        for i in range(len(s)):
            char = s[i]

            while char in char_set:
                char_set.remove(s[left])
                left += 1

            char_set.add(char)
            result = max(result, len(char_set))

        return result
```

維護一個 set, 以及 left pointer, 用一個 for 迴圈當作他的 right pointer, 如果當下的 right char 已經在 set 裡，用 while 不斷的把 left char 移走，直到沒有 right char，這樣就可以把它加入到 set，然後更新最長的長度。

time: 兩個指標都往右走，不會返回，最多移動 2n 步，因此是 O(n)
space: char_set 最多存 n 個步重複字元，因此是 O(n)

==如果題目有限制只能是 ASCII 字元（256種）或英文小寫（26種），space 可以是 O(1)==，代表大小有上限
