## Callback function : callback 是一種可以當作是函式參數一樣被帶進其他函式的函式，用途在於控制函示的執行順序，但當程式變得複雜時，容易變成 callback hell。

## Promise : Promise 是一種語法，用來專門解決非同步的問題，將函示的結果可以用以預定的方式執行。

## async、await : async 用來定義一個函示本體為非同步，但讓函示內的非同步程式以同步的方式處理，await 則是暫停非同步(promise)的進行，直到進入 resolve 或 reject 並回傳再接著下去執行。

Async functions always return a promise，if the function returns a value，the promise will be resolved with that value

- javascript fetch method : fetch is going to resolve the promise as soon as it receives the header from the API，so it doesn't wait all the data is back。We have to use res.json() method that returns a promise which resolves with the result of parsing the body text as JSON.
