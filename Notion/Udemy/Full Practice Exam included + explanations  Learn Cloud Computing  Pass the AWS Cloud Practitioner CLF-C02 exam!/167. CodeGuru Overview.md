## **1. What is Amazon CodeGuru?**

Amazon CodeGuru is a **machine-learning powered developer tool** that provides:

1. **Automated code reviews** → CodeGuru Reviewer
2. **Application performance profiling** → CodeGuru Profiler

It helps detect bugs, improve performance, and reduce cost—both **before deployment** and **in production**.

---

## **2. CodeGuru Reviewer**

### **Purpose**

Provides **static code analysis** for code stored in repositories like:

- GitHub
- Bitbucket
- CodeCommit

### **What it detects**

- Critical issues and bugs
- Security vulnerabilities
- Resource leaks (e.g., open connections)
- Input validation issues
- Inefficient or dangerous coding patterns

### **How it works**

- Uses ML trained on:
    - Thousands of open-source repos
    - Amazon’s internal repositories
- Generates actionable recommendations on specific lines of code

### **Languages Supported**

- Java
- Python

---

## **3. CodeGuru Profiler**

### **Purpose**

Analyzes **runtime performance** of applications in:

- Production
- Pre-production
- On-premises or AWS

### **What it does**

- Identifies high CPU usage lines (“hot paths”)
- Detects slow or inefficient parts of the application
- Highlights memory-heavy objects (heap analysis)
- Provides anomaly detection for unusual app behavior
- Helps reduce compute costs (performance tuning → fewer CPU cycles)

### **Use Cases**

- Optimize Lambda / EC2 / ECS costs
- Understand why an application is slowing down
- Reduce latency
- Find performance bottlenecks early

Profiler introduces **minimal overhead** on the running application.

# ⭐ **CLF-C02 Exam Tips**

### **1. Reviewer vs Profiler (非常重要)**

| Feature | CodeGuru Reviewer | CodeGuru Profiler |
| --- | --- | --- |
| When used | During code review (pre-deployment) | Runtime (production or staging) |
| Purpose | Static analysis, detect bugs & security issues | Improve performance & reduce cost |
| Detects | Bad code patterns, leaks, vulnerabilities | CPU bottlenecks, memory issues, slow code |

Exam keyword mapping:

- “static code analysis” → Reviewer
- “runtime performance optimization” → Profiler
- “reduce CPU cost” → Profiler
- “detect security bugs in pull requests” → Reviewer

---

### **2. Integration**

- Reviewer integrates with GitHub, CodeCommit, Bitbucket
- Profiler works for apps on AWS **or on-premises**
- Both support Java & Python (CLF 不會考更細規格)
