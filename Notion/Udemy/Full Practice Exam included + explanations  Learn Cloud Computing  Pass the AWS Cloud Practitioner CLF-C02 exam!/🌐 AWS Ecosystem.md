## 🎯 What is the AWS Ecosystem?

The **AWS Ecosystem** includes **learning resources, tools, partners, support, and marketplaces** that help customers **build, operate, and optimize** workloads on AWS.

> Exam mindset:
> **AWS ≠ only services → includes learning, support, partners, and solutions**

---

## 📚 Free Learning & Knowledge Resources (Exam-Friendly)

### AWS Blog

- New feature releases
- Best practices & deep dives
- Cost & performance optimization tips

---

### Community Forums

- Ask questions
- Learn from other AWS users
- Community-driven knowledge

---

### Whitepapers & Guides

- In-depth architectural guidance
- Official AWS best practices
- Common exam reference material

---

### AWS Solutions Library (formerly Quick Starts)

- **Pre-built, vetted architectures**
- Real-world use cases (e.g., live streaming)
- Launch directly via **CloudFormation**

> Exam keyword:
> **Vetted solutions + Infrastructure as Code**

## 🛠️ AWS Support Plans (VERY IMPORTANT FOR EXAM)

| Plan | Key Features |
| --- | --- |
| **Developer** | Business-hours email support |
| **Business** | 24/7 phone, chat, email |
| **Enterprise** | TAM + fastest response |

---

### ⏱️ Support Response Time (Know the Order)

### Developer

- General guidance: < 24 hours
- System impaired: < 12 hours

### Business

- System impaired: < 4 hours
- Production system down: < 1 hour

### Enterprise ⭐

- Dedicated **Technical Account Manager (TAM)**
- Concierge billing support
- Business-critical system down: **< 15 minutes**

> Exam hint:
> **Fastest response + TAM = Enterprise Support**

## 🏪 AWS Marketplace (Exam Favorite)

### What is it?

A **digital catalog** of third-party software integrated with AWS billing.

---

### What You Can Buy

- Custom AMIs
- SaaS products
- Containers
- CloudFormation templates

---

### Key Benefits

- Integrated into **AWS billing**
- Easy vendor procurement
- Can also **sell your own solutions**

> Exam keyword:
> **Third-party software billed through AWS**

## 🤝 Professional Services & AWS Partner Network (APN)

### AWS Professional Services

- AWS experts work **with your team**
- Help with migrations & architecture

---

### AWS Partner Network (APN)

A network of **AWS-approved partners**

### APN Partner Types

- **Technology Partners** → software & infrastructure
- **Consulting Partners** → design & implementation
- **Training Partners** → AWS education

---

### Competency Program

- Recognizes partners with:
    - Proven expertise
    - Customer success
    - Specialized solutions

---

### Navigate Program

- Helps partners improve AWS skills
- Partner enablement
