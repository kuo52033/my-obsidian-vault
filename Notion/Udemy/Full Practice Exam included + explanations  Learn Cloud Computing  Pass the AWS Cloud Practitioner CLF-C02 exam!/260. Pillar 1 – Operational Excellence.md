## **What is Operational Excellence?**

**Operational Excellence** is the ability to:

- **Run and monitor systems** effectively
- **Deliver business value**
- **Continuously improve processes and procedures**

📌 關鍵字（考試常見）

> Run, Monitor, Improve, Deliver Business Value

---

## **Design Principles of Operational Excellence (必背)**

### **1. Perform Operations as Code**

- Everything should be automated
- Use **Infrastructure as Code (IaC)**
- AWS Example:
    - **AWS CloudFormation**

📌 *No manual configuration*

---

### **2. Make Frequent, Small, Reversible Changes**

- Deploy often
- Small changes = lower risk
- Easy rollback if something breaks

❌ Big changes every few months

✅ Small changes frequently

---

### **3. Refine Operations Procedures Frequently**

- Update runbooks
- Train team regularly
- Everyone understands current procedures

---

### **4. Anticipate Failure & Learn from It**

- Failures are expected
- Learn from incidents
- Improve processes continuously

📌 *Fail → Learn → Improve*

---

### **5. Use Managed Services**

- Reduce operational burden
- AWS handles:
    - Patching
    - Scaling
    - Availability

---

### **6. Implement Observability**

Gain **actionable insights** into:

- Performance
- Reliability
- Cost

📌 *You can’t improve what you can’t observe*

## **Operational Excellence in AWS (Conceptual Mapping)**

> ⚠️ Exam does NOT require memorizing all services below
> This is for understanding the ecosystem

---

### **Prepare**

Goal: Be ready before production

- **Runbooks**
- **Infrastructure standards**
- Mock deployments

AWS Services:

- CloudFormation → IaC
- AWS Config → Compliance & configuration evaluation

---

### **Operate**

Goal: Run systems efficiently

- Automate operations
- Avoid manual changes
- Monitor everything

AWS Services:

- CloudFormation → Automated deployments
- CloudTrail → Track API calls
- CloudWatch → Metrics, logs, alarms
- X-Ray → Trace requests & debug issues

---

### **Evolve**

Goal: Improve continuously

- Fast iteration
- CI/CD pipelines
- Small & frequent deployments

AWS Services:

- CodeCommit
- CodeBuild
- CodeDeploy
- CodePipeline
