**Amazon SageMaker** is a fully managed service for **developers and data scientists** to build, train, and deploy **custom machine learning models**.

Unlike other ML services（Rekognition, Comprehend, Transcribe…），which solve a **specific ML task**, SageMaker is for building **your own ML models** end-to-end.

---

# **✨ What SageMaker Provides**

SageMaker covers the **full machine learning lifecycle**:

### **1. Data Labeling**

- Annotate and prepare datasets
- Built-in human labeling workflows via **Ground Truth**

### **2. Model Building**

- Managed Jupyter notebooks for experimentation
- Pre-built ML algorithms
- Bring-your-own frameworks (TensorFlow, PyTorch, etc.)

### **3. Model Training**

- Train at scale on fully managed ML compute
- Distributed training support
- Automatic model tuning (hyperparameter optimization)

### **4. Model Deployment**

- One-click deployment to scalable endpoints
- Real-time inference or batch transform
- Managed hosting infrastructure

---

# **🧠 Why Use SageMaker?**

- No need to manage servers for ML training
- Integrated workflows (data → model → deployment)
- Supports custom ML, not just pre-trained functionality
- Efficient for large-scale enterprise ML development

---

# **📘 Simple Example (Exam Understanding)**

To predict a student's certification exam score, SageMaker would help you:

1. **Collect data** (experience, study time, results)
2. **Label it** (associate input with actual exam score)
3. **Train a model** to learn the relationship
4. **Tune the model** for higher accuracy
5. **Deploy the model** to make predictions on new students

All steps can be accomplished **inside SageMaker**.

# **💡 Exam Tips（CLF-C02 必考）**

- SageMaker = **Build / Train / Deploy custom ML models**
- Not used for:
    - Image recognition → Rekognition
    - NLP sentiment → Comprehend
    - Translation → Translate
    - Speech-to-text → Transcribe
    - Chatbots → Lex
- SageMaker is for **data scientists + ML pipelines**, not turnkey ML tasks.
