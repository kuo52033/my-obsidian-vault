## What is AWS Wavelength?

AWS Wavelength deploys **AWS infrastructure inside 5G telecom providers' data centers**,

so applications can run **extremely close to mobile users**.

This enables **ultra-low latency** for applications accessed over 5G networks.

Whenever you see **5G + ultra-low latency** in exam questions → **the answer is AWS Wavelength**.

---

## How It Works

A Wavelength Zone is located **inside the 5G provider’s network**.

You can deploy AWS resources directly into that zone:

- **EC2 instances**
- **EBS volumes**
- **VPC subnets**

Users on the telecom’s 5G network connect to these resources **without leaving the carrier network**, giving *sub-10 ms latency*.

When needed, Wavelength Zones are still connected to an AWS **parent Region**, allowing access to:

- DynamoDB
- RDS
- S3
- Other AWS services

Traffic that must go back to AWS takes a secure, optimized path.

---

## Architecture Overview

**Mobile Device (5G) → Telecom 5G Network → Wavelength Zone → Parent AWS Region (optional)**

- **Local processing** in Wavelength = ultra low latency
- **Deeper AWS services** still reachable in Region

---

## Key Benefits

- **Ultra-low latency** for mobile and 5G applications
- Keeps traffic **inside the CSP (carrier) network** for fast local processing
- Runs AWS compute/storage **at the 5G edge**
- Seamlessly integrates with your AWS VPC
- **No additional agreements or special costs** beyond standard AWS usage pricing

## Use Cases (Exam-important)

AWS Wavelength is ideal for applications where latency is critical:

- **Smart cities / IoT at scale**
- **Connected vehicles**
- **AR / VR**
- **Real-time gaming**
- **Interactive live video**
- **Machine-learning assisted diagnostics**
- **Industrial automation**
- **Drones, robotics, autonomous systems**

Essentially: **anything requiring real-time responses to 5G mobile devices**.

---

## When to Use Wavelength (Exam Tips)

Choose **AWS Wavelength** when:

- You need **ultra-low latency** for 5G mobile users.
- You require compute **inside the telecom network**.
- You need **nearby processing** for heavy real-time workloads.

Do **not** choose Local Zones, Outposts, or Global Accelerator for 5G use cases—

**Wavelength** is the only service designed specifically for 5G edge deployment.
