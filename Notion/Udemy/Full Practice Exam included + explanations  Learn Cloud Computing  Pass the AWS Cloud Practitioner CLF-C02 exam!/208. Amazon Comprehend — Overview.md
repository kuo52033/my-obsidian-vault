**Amazon Comprehend** is AWS’s **Natural Language Processing (NLP)** service.

It uses machine learning to extract insights, structure, and meaning from unstructured text.

👉 **Exam keyword:** If you see **NLP / sentiment / key phrases / text analytics → Amazon Comprehend**

---

## **✨ Key Capabilities**

Comprehend can automatically analyze text and extract:

### **1. Language Detection**

Detect the source language of the text.

### **2. Key Phrase Extraction**

Identify important phrases, entities, or concepts.

### **3. Entity Recognition**

Recognize:

- People
- Places
- Events
- Brands
- Organizations

### **4. Sentiment Analysis**

Determine whether a text is:

- Positive
- Negative
- Neutral
- Mixed

### **5. Syntax & Tokenization**

Break text into tokens and parts of speech (verbs, nouns, adjectives…).

### **6. Topic Modeling**

Automatically group large volumes of documents into topics.

---

## **🧠 What Comprehend Is Used For**

- Customer feedback analysis (emails, reviews, support tickets)
- Grouping articles by topic
- Extracting business insights from unstructured text
- Detecting sentiment trends
- Text classification workflows

---

## **💡 Exam Tips**

- Comprehend = **NLP**
- Use it for **sentiment analysis**, **entity recognition**, **topic modeling**, or **language detection**
- Not used for translation (→ Translate)
- Not used for speech transcription (→ Transcribe)
- Not used for image text extraction (→ Textract)
