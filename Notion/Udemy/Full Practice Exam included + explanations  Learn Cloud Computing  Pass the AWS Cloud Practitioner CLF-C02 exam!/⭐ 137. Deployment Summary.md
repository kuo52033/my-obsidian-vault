## Infrastructure Deployment Tools

### **CloudFormation**

- AWS-native **Infrastructure as Code (IaC)**.
- Declarative templates (YAML/JSON) to define AWS resources.
- Supports almost all AWS services.
- Enables **repeatable, consistent deployments** across regions and accounts.
- Ideal for complex architectures and automated provisioning.

---

### **Elastic Beanstalk**

- A **Platform as a Service (PaaS)**.
- Focused on **developer-friendly application deployment**.
- Supports certain languages & Docker.
- Automatically provisions:
    - Load balancers
    - EC2 instances
    - Auto Scaling
    - Optional RDS
- You provide code → Beanstalk handles infra + deployment.

---

### **CodeDeploy**

- Automates application deployments **to servers**.
- Supports:
    - EC2 instances
    - On-premises servers
- Hybrid service (AWS + on-prem).
- Great for updating apps from version 1 → version 2 with minimal downtime.

---

### **Systems Manager (SSM)**

- Hybrid service for **managing EC2 + on-prem servers** at scale.
- Provides:
    - Automated patching
    - Remote command execution
    - Configuration management
- Centralized operational control.

---

## Developer & CI/CD Services

### **CodeCommit**

- Private Git-based code repository hosted in AWS.
- Secure, scalable, fully managed.
- Works like GitHub/Bitbucket but integrated with AWS IAM.

---

### **CodeBuild**

- Fully managed, **serverless build service**.
- Compiles code, runs tests, produces build artifacts.
- Pay only for build time.

---

### **CodeDeploy** (Developer Side Note)

- Also relevant in CI/CD flows for automated deployments.

---

### **CodePipeline**

- Orchestrates full **CI/CD pipelines**:
    - Source → Build → Test → Deploy
- Integrates with:
    - CodeCommit
    - CodeBuild
    - CodeDeploy
    - CloudFormation
    - Beanstalk
    - GitHub
    - Third-party plugins

Ideal for automating software release workflows.

---

### **CodeArtifact**

- Secure, scalable **artifact & package dependency repository**.
- Works with:
    - npm
    - pip
    - Maven
    - NuGet
    - yarn
- Stores shared libraries and dependencies for teams.

---

### **AWS CDK (Cloud Development Kit)**

- Define AWS infrastructure **using real programming languages**:
    - TypeScript, JavaScript, Python, Java, .NET
- CDK code → compiles into CloudFormation templates.
- Lets developers share logic and reuse structures using code patterns.
