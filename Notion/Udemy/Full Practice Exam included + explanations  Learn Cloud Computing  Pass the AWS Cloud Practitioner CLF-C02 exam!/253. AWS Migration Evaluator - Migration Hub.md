## **Purpose**

Build a **data-driven business case** for migrating to AWS.

> Focus: Cost, baseline, and business justification, not actual migration.

---

## **What Problem It Solves**

Before migrating, organizations need to answer:

- What workloads do we run today?
- How much do they cost on-premises?
- What will they cost on AWS?
- Is migration worth it from a business perspective?

AWS Migration Evaluator answers these questions.

---

## **How It Works**

### **1. Baseline Current Environment**

- Install **Agentless Collector**
    - Broad discovery of on-prem infrastructure
    - No agents on servers
- OR use **Data Import**
    - Upload existing inventory data using AWS templates

---

### **2. Analyze Current State**

- On-prem infrastructure footprint
- Server inventory
- Dependencies
- Current cost structure

---

### **3. Define Target State**

- AWS architecture options
- Right-sizing recommendations
- Cost optimization opportunities

---

### **4. Build Migration Business Case**

- Customized cost insights
- Migration cost vs current cost
- ROI and savings analysis
- Optional **AWS expert guidance**

---

## **Key Output**

- Financial and operational justification for migration
- Clear cost comparison: **On-prem vs AWS**

---

## **Exam Keywords**

- “Business case”
- “Cost analysis before migration”
- “Data-driven migration decision”

# **254. AWS Migration Hub**

## **Purpose**

A **central hub** to **discover, plan, track, and manage migrations** to AWS.

> Focus: Visibility & coordination, not cost estimation.

---

## **What Migration Hub Does**

- Centralizes migration data
- Tracks progress across multiple tools
- Accelerates migration workflows

---

## **Key Capabilities**

### **Central Inventory**

- Servers
- Applications
- Dependencies
- Migration status

---

### **Migration Tracking**

- Track lift-and-shift and modernization
- Monitor progress across accounts and regions

---

### **Integration with Migration Services**

- **Application Discovery Service**
- **Application Migration Service (MGN)**
- **Database Migration Service (DMS)**

---

### **Migration Hub Orchestrator**

- Pre-built workflows
- Designed for **enterprise apps**
    - SAP
    - Microsoft SQL Server
- Saves time and reduces complexity

---

### **Modernization Support**

- Incremental refactoring
- Strategy recommendations
- Right-sizing insights

---

## **Exam Keywords**

- “Central location for migration”
- “Track migration progress”
- “Multiple migration tools in one place”

➡️ **AWS Migration Hub**
