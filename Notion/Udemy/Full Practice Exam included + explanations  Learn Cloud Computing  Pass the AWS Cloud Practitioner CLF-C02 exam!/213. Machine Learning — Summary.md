This section covers all AWS ML services that may appear in the exam.

Memorize the **one-line purpose** of each service — this is exactly how the exam tests them.

---

## **🖼️ Rekognition — Image & Video Analysis**

- Detect faces, objects, scenes
- Facial analysis / face search
- Celebrity recognition
- Content moderation

---

## **🎤 Transcribe — Speech → Text**

- Convert audio to text
- Used for subtitles & call transcripts
- Supports PII redaction + multi-language detection

---

## **🔊 Polly — Text → Speech**

- Generate lifelike speech from text
- Neural TTS, multiple voices and languages
- Opposite of Transcribe

---

## **🌍 Translate — Language Translation**

- Real-time or batch translation
- Used to localize content for global users

---

## **💬 Lex — Chatbots / Voicebots**

- Powers Alexa
- ASR (speech-to-text) + NLU (intent understanding)
- Build conversational interfaces

### **Lex + Connect**

- Connect = Cloud contact center
- Combined → intelligent call center automation

---

## **🔎 Comprehend — NLP (Natural Language Processing)**

- Sentiment analysis
- Key phrase extraction
- Entity recognition
- Topic modeling
- Language detection

---

## **🧪 SageMaker — Build, Train, Deploy ML Models**

- Full ML pipeline for developers & data scientists
- Custom ML models (not pre-built tasks)
- End-to-end: labeling → training → tuning → deployment

---

## **📚 Kendra — Intelligent Document Search**

- Enterprise search engine
- Natural language queries
- Pulls answers from PDFs, docs, HTML, FAQs, etc.

---

## **🎯 Personalize — Recommendation Engine**

- Real-time personalized product/content recommendations
- Same tech used by Amazon.com
- No ML expertise required

---

## **📄 Textract — Extract Text from Documents**

- OCR + handwriting recognition
- Extracts forms, tables, key-value pairs
- Used for invoices, IDs, tax forms
