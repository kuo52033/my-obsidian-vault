# **Multi-Account Management**

- Use AWS Organizations to manage multiple accounts.
- Use Service Control Policies (SCPs) to restrict what accounts and users can do.
- Use AWS Control Tower for automated, secure multi-account setup following best practices.

# **Security Best Practices**

- Enable MFA on all critical accounts (especially root).
- Follow least privilege IAM principles.
- Create and enforce password policies + password rotation.
- Use AWS Config to record configuration changes and check compliance.
- Use CloudTrail to log API activity across all regions.
- Send logs (service logs, access logs) to S3 or CloudWatch Logs, preferably into a centralized logging account.

# **Infrastructure & Deployment**

- Use CloudFormation to deploy and manage infrastructure across multiple regions/accounts.
- Use Resource Tags & Cost Allocation Tags for:
    - Billing
    - Organization
    - Automation
    - Resource grouping
- Use Service Catalog to allow users to launch pre-approved resources via CloudFormation templates.

# **Account Recovery & Incident Response**

If an AWS account becomes compromised:

1. Change the root user password immediately.
2. Delete or rotate all passwords and access keys.
3. Contact AWS Support.

# **233. Billing Summary**

# **Cost Optimization Tools**

**1. Compute Optimizer**

- Uses ML to recommend optimal EC2, ASG, EBS, and Lambda configurations.
- Helps reduce cost and improve performance.

**2. Savings Plans**

- Commit to spending a fixed amount ($/hour) for 1 or 3 years.
- More flexible alternative to Reserved Instances.

# **Cost Estimation & Forecasting**

**3. AWS Pricing Calculator**

- Estimate monthly cost before building infrastructure.

**4. Billing Dashboard**

- High-level cost overview (month-to-date cost, forecast, top services).

**5. Cost Explorer**

- Visualize cost over time.
- Analyze cost by service, region, account, or tag.
- Forecast costs up to 12 months (exam keyword!)
- Provides Savings Plan recommendations.

# **Detailed Cost Tracking**

**6. Cost Allocation Tags**

- Identify costs by:
    - Project
    - Team
    - Environment
    - Cost Center
- Enable user-defined or AWS-generated tags.

**7. Cost & Usage Reports (CUR)**

- Most detailed, granular billing dataset.
- Export to S3.
- Analyze using Athena, Redshift, or QuickSight.

# **Cost Monitoring & Alerts**

**8. Billing Alarms (CloudWatch)**

- Trigger alarms based on actual spend.
- Only available in us-east-1.

**9. AWS Budgets**

- More advanced than billing alarms.
- Alert on:
    - Actual cost
    - Forecasted cost
    - Usage
    - RI & Savings Plan utilization
- Supports SNS notifications and automation.

**10. Cost Anomaly Detection**

- Uses machine learning to detect unusual spending spikes.
- Sends alerts (daily, weekly, or immediate).

**11. Service Quotas**

- Monitor AWS quota limits and set alarms.
- Request quota increases directly.
