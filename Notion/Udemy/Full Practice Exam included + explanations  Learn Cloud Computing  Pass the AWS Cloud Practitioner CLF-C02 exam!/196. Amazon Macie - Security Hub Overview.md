Amazon Macie is a **fully managed data security and privacy service** that uses **machine learning + pattern matching** to discover and protect **sensitive data in S3**.

---

# **1. What Macie Does**

Macie scans **S3 buckets** to automatically detect:

- **PII (Personally Identifiable Information)**
- Financial data
- Credentials
- Sensitive text patterns (emails, names, phone numbers, addresses)

Macie classifies and labels sensitive data to help prevent data leaks.

---

# **2. How Macie Works**

1. Enable Macie
2. Choose S3 buckets to scan
3. Macie analyzes objects
4. Findings sent to **EventBridge**
5. You can trigger:
    - SNS notifications
    - Lambda remediation
    - Security Hub aggregation

---

# **3. Key Characteristics**

- Only scans **S3**, not EC2/RDS/EBS/etc.
- One-click enablement
- Uses ML to detect sensitive data
- Integrates with Security Hub & EventBridge
- Helps with compliance (GDPR, HIPAA, PCI)

# **197. AWS Security Hub Overview**

AWS Security Hub is a **centralized security management and compliance dashboard**.

It aggregates findings from multiple AWS security services and partner tools.

---

# **1. What Security Hub Does**

- Collects security alerts from multiple services
- Runs automated compliance checks (e.g., CIS AWS Foundations Benchmark)
- Provides a **single dashboard** across multiple accounts
- Supports automation via EventBridge

---

# **2. Services That Feed Into Security Hub**

Security Hub aggregates findings from:

- **GuardDuty** (threat detection)
- **Inspector** (vulnerability scanning)
- **Macie** (PII detection)
- **Config** (compliance)
- **Firewall Manager**
- **IAM Access Analyzer**
- **Systems Manager**
- **AWS Health**
- Partner security tools (e.g., Splunk, CrowdStrike)

### Important:

> AWS Config must be enabled for Security Hub to operate.

---

# **3. Architecture Summary**

Security Hub ← Findings from GuardDuty, Macie, Inspector, Config, etc.

↓

Security Hub Dashboard (multi-account view)

↓

EventBridge Events → SNS / Lambda automation

↓

Investigation via **Amazon Detective**

---

# ⭐ **CLF-C02 Exam Tips — Security Hub**

### **3. Need automated compliance checks?**

✔ Security Hub
