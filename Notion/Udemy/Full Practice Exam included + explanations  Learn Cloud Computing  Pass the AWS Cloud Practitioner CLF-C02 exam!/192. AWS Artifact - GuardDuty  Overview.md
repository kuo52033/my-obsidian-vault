AWS Artifact is **not a compute service**, but a **self-service portal** for accessing AWS compliance documentation.

---

## **1. What AWS Artifact Provides**

### **A. Artifact Reports**

Download official AWS compliance reports, such as:

- **SOC 1 / SOC 2 / SOC 3**
- **ISO certifications**
- **PCI reports**
- **Third-party audit reports**

These documents help your organization prove compliance for internal and external audits.

---

### **B. Artifact Agreements**

Review and accept legal agreements, such as:

- **HIPAA BAA (Business Associate Addendum)**
- Other regulatory compliance agreements

These agreements apply to:

- Individual AWS accounts
- AWS Organizations accounts

---

## **2. Key Characteristics**

- **Global service**
- **Free to use**
- No technical deployment — purely documentation & compliance management
- Supports audit readiness for security, governance, and regulatory frameworks

---

# ⭐ **CLF-C02 Exam Tips — AWS Artifact**

> “Where can you download AWS compliance reports (SOC, PCI, ISO)?”
> ✔ **AWS Artifact**

> “Where do you accept HIPAA BAA agreements?”
> ✔ **AWS Artifact**

> “Does Artifact provide operational AWS services?”
> ❌ No — It only provides documents & agreements.

# **193. GuardDuty Overview**

Amazon GuardDuty is a **threat detection service** that continuously monitors AWS environments using machine learning and anomaly detection.

---

# **1. What GuardDuty Does**

GuardDuty analyzes multiple data sources to detect:

- Unauthorized access
- Malicious API calls
- Compromised EC2 instances
- Cryptocurrency mining activity
- Unusual data access patterns
- Suspicious network behavior

---

# **2. Data Sources GuardDuty Monitors**

### **Always enabled:**

- **CloudTrail Events**
    - Unusual API calls
    - Unauthorized deployments
- **VPC Flow Logs**
    - Unusual network traffic
    - Strange IP communication patterns
- **DNS Logs**
    - EC2 instances sending encoded or suspicious DNS queries

### **Optional enhanced protections:**

- S3 data events (GetObject, PutObject, DeleteObject)
- EKS Audit Logs
- RDS & Aurora login activity
- EBS volume monitoring
- Lambda network activity

GuardDuty continues adding more over time.

---

# **3. How Alerts Work**

GuardDuty generates **security findings**, which can be sent to:

- **Amazon EventBridge**
    - Trigger Lambda remediation
    - Trigger SNS notifications
    - Integrate into SIEM / SOAR workflows

---

# **4. Characteristics of GuardDuty**

- One-click enablement
- No infrastructure to install
- 30-day free trial
- Regional service (enable per region)

---
