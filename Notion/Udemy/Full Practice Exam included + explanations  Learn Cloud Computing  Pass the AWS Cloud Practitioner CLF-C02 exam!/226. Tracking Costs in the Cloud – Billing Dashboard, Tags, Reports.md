## **1. Billing Dashboard**

A high-level view of your AWS monthly spending.

### **What it shows**

- **Month-to-date cost**
- **Last month cost**
- **Forecasted cost**
- **Cost breakdown by service / region / account**

### **Use cases**

- Quick overview of current spending
- Initial investigation when the bill looks higher than expected

---

## **2. Cost Allocation Tags**

Tags help categorize and track cloud spending based on your business structure.

### **Tag Types**

| Type | Prefix | Description |
| --- | --- | --- |
| **AWS-generated tags** | `aws:` | Automatically created (e.g., *aws:createdBy*) |
| **User-defined tags** | `user:` | Custom tags: *environment*, *team*, *cost-center*, *app*, etc. |

### **Purpose**

- Organize resources
- Attribute cost to **departments / teams / applications**
- Generate detailed cost reports
- Build **Resource Groups**

### **Example Tags**

- `Environment = dev / prod`
- `Team = backend`
- `Owner = tim`
- `CostCenter = IT-001`

### **Resource Groups & Tag Editor**

- Tag Editor lets you **search, edit, and apply tags** across resources
- Resource Groups allow you to view resources sharing the same tags
→ Useful for operational management and cost visibility

## **3. Cost and Usage Reports (CUR)**

The **most detailed billing report** AWS provides.

### **Key characteristics**

- Includes **all cost & usage data**, metadata, RI/Savings Plans details
- Delivered to an **S3 bucket**
- Integrates with **Athena**, **Redshift**, or **QuickSight** for analysis

### **Typical Use Cases**

- Deep investigation into billing anomalies
- Building dashboards for finance teams
- Understanding hourly/daily cost at resource level

---

## **4. Cost Explorer**

A visual tool to view, analyze, and forecast AWS spending.

### **Features**

- Monthly, daily, or hourly cost visualization
- Grouping by service, region, usage type, or tag
- Resource-level breakdown
- Identify cost trends
- **Forecast up to 12 months** (common exam question)
- Provides **Savings Plan recommendations**

### **When to use**

- To track real-time spending trends
- To find unexpected high-cost services
- To plan budgets using forecast

---

## **5. Data Exports**

Allows exporting cost and usage datasets in various formats.

### **Options**

- Choose **CSV or Parquet**
- Select columns and granularity (daily/hourly)
- Store exports in **S3**
- Feed into **Athena** for SQL queries
