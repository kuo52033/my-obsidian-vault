## **What is Amazon AppStream 2.0?**

- **Fully managed application streaming service**
- Streams **individual desktop applications** to users
- Applications are delivered **through a web browser**

---

## **Core Idea**

> Stream applications, not desktops

Users do **NOT** get a full virtual desktop.

They only access **specific applications** on demand.

---

## **How AppStream 2.0 Works (High Level)**

1. Admin uploads and configures an application (e.g. Blender, Photoshop)
2. AWS runs the app on a managed instance
3. User opens a **web browser**
4. Application UI is streamed to the browser in real time

---

## **Key Characteristics**

- **Application-focused** (not a full OS desktop)
- **Browser-based access**
- **No client software required**
- **Works on any device** with a web browser
- **Per-application instance sizing** (CPU / RAM / GPU configurable)
- **Fully managed by AWS**

---

## **Typical Applications**

- 3D modeling (Blender)
- Design tools (Photoshop)
- Development tools (Eclipse)
- Office apps (OpenOffice)
- Any desktop app that needs high compute but easy access

---

## **AppStream 2.0 vs WorkSpaces (Exam Favorite)**

| Feature | AppStream 2.0 | WorkSpaces |
| --- | --- | --- |
| What is streamed | **Single application** | **Full desktop (VDI)** |
| Access method | Web browser | Desktop client / RDP |
| User experience | App-only | Full Windows/Linux desktop |
| Device compatibility | Any device with browser | Requires WorkSpaces client |
| Use case | App delivery | Virtual desktops |
| Infrastructure | Per app instance | Per user desktop |
| Session type | On-demand | Always-on or on-demand |

---

## **When to Use AppStream 2.0**

Use AppStream 2.0 when:

- Users need **specific applications only**
- You want **browser-based access**
- You don’t want to manage desktops
- Applications require **high CPU / GPU**
- Users use **many different devices**

---

## **What AppStream 2.0 Is NOT**

- ❌ Not a virtual desktop (that’s **WorkSpaces**)
- ❌ Not application hosting for backend services
- ❌ Not OS-level user customization
