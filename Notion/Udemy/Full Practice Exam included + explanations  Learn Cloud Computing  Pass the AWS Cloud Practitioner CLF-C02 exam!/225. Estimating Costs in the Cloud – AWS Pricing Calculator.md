## **1. What Is the AWS Pricing Calculator?**

The AWS Pricing Calculator is a **free web-based tool** that helps estimate the cost of AWS services before you deploy them.

🔗 URL: [https://calculator.aws](https://calculator.aws/)

It allows you to:

- Model **architectures** and estimate monthly cost
- Compare pricing options (On-Demand vs Reserved Instances vs Savings Plans)
- Adjust configuration (instance type, storage, data transfer, etc.)
- Generate **shareable** and **downloadable** cost reports

---

## **2. Why Use the Calculator?**

### **Key Benefits**

- **Accurate cost prediction** before building
- Helps with **budget planning**
- Helps compare multiple architecture options
- Frequently used in **enterprise planning** and **migration projects**

---

## **3. What You Can Estimate**

Examples of AWS services you can include in your estimate:

- EC2 (instances, storage, load balancers)
- RDS (instance hours, storage, backups)
- S3 (storage classes, lifecycle transitions, data transfer)
- Lambda (requests + compute time)
- DynamoDB (read/write capacity, on-demand usage)
- CloudFront (data transfer + requests)
- EKS / ECS cost modeling
- VPC data transfer costs

Each service allows fine-grain configuration to reflect real usage.

---

## **4. Typical Workflow**

1. Open **AWS Pricing Calculator**
2. Click **Create estimate**
3. Add services (EC2, S3, Lambda, RDS…)
4. Configure:
    - Region
    - Usage patterns
    - Instance size / storage / network
5. Review summary
6. Export/share estimate (PDF or link)

---

## **5. Exam Tips**

- *“Which AWS tool helps estimate the cost of a planned architecture?”*
→ **AWS Pricing Calculator**
- *“Which tool lets you model costs before deploying AWS resources?”*
→ **AWS Pricing Calculator**
- Do not confuse with:
    - **Cost Explorer** → analyzes **existing** spending
    - **Billing Dashboard** → shows actual bills
    - **Budgets** → alerts when cost exceeds threshold
    - **Trusted Advisor** → gives cost-saving recommendations

👉 Pricing Calculator = **Pre-deployment cost estimation**
