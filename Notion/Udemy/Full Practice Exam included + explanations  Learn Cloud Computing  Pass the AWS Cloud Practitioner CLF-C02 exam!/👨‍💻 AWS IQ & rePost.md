## 🔧 AWS IQ

### 🎯 What is AWS IQ? (Exam-friendly)

**AWS IQ** is a platform to **find and hire AWS-certified third-party experts** for **on-demand project work**.

> Think of it as:
> **AWS-native freelancer platform**

---

### 🧩 Key Features

- AWS-certified professionals
- On-demand project engagement
- Built-in:
    - Video conferencing
    - Contract & milestone management
    - Secure collaboration
    - **Integrated AWS billing**

---

### 🔄 How AWS IQ Works (Customer Side)

```plain text
Submit project request
   ↓
Receive proposalsfrom experts
   ↓
Review experience, rate, timeline
   ↓
Select expert
   ↓
Grant secure AWSaccess
   ↓
Approve milestones
   ↓
Charged directlyto AWS bill


```

---

### 🧠 Expert Side (FYI)

- Create profile (certifications, experience)
- Submit proposals
- Work securely
- Get paid after milestones are approved

---

### 🧠 CLF-002 Exam Notes (AWS IQ)

- Used to **hire experts**, not ask questions
- Paid work
- Billing goes through **AWS account**
- For **project delivery**, not general Q&A

---

### 🧩 One-Sentence Exam Answer

> AWS IQ allows customers to find, engage, and pay AWS-certified experts for on-demand project work.

## 💬 AWS re:Post

### 🎯 What is AWS re:Post?

**AWS re:Post** is an **AWS-managed community Q&A forum** for asking and answering **technical AWS questions**.

> Replacement for the old AWS Forums

---

### 🧩 Key Characteristics

- Community-driven
- Expert-reviewed answers
- Similar to Stack Overflow
- Earn reputation by contributing
- Join interest-based groups

---

### 🧠 Premium Support Integration

- If **no community answer**:
    - Question is **escalated to AWS Support engineers**
    - Only for **Premium Support customers**

---

### 🚫 What re:Post is NOT for (Very Exam-Important)

- ❌ Time-sensitive issues
- ❌ Production outages
- ❌ Proprietary or confidential actions

> Exam phrase:
> **Not for urgent or time-critical support**

---

### 🧠 CLF-002 Exam Notes (re:Post)

- Used for **knowledge sharing**
- Free community resource
- Not a replacement for AWS Support plans

---

### 🧩 One-Sentence Exam Answer

> AWS re:Post is a community-driven Q&A service for AWS technical questions with expert-reviewed answers.

---

## 🧠 AWS IQ vs AWS re:Post (Exam Gold)

| Feature | AWS IQ | AWS re:Post |
| --- | --- | --- |
| Purpose | Hire experts | Ask questions |
| Cost | Paid (project-based) | Free |
| Billing | AWS bill | No billing |
| Urgent issues | ❌ No | ❌ No |
| Project work | ✅ Yes | ❌ No |
| Community Q&A | ❌ No | ✅ Yes |
