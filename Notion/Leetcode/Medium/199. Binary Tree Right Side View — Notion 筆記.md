## ⭐ **難度：Medium (5/10)**

## 📌 **面試常考程度：8/10**

這題是 **Binary Tree BFS / DFS 的經典題**，

常被用來考：

- Tree 的層級遍歷（Level Order Traversal）
- 是否理解「每一層只取一個值」
- BFS vs DFS 的不同寫法

---

## 🎯 **題意**

給定一棵二元樹，

請回傳「**從右邊看這棵樹時，能看到的節點值**」。

換句話說：

👉 **每一層最右邊的節點**

---

## 💡 **核心觀念**

### 關鍵轉換

> 「右側視角」
> ⟹ 「**每一層的最後一個節點**」

所以這題可以轉成：

- **Level Order Traversal（BFS）**
- 每一層只記錄 **最後一個節點的值**

---

## 🧠 **你目前的解法（BFS，正確）**

```python
from collections import deque

class Solution:
    def rightSideView(self, root: Optional[TreeNode]) -> List[int]:
        if not root:
            return []

        stack = [root]
        result = []

        while len(stack):
            dq = deque(stack)
            stack = []          

            while len(dq):
                n = dq.popleft()

                if len(dq) == 0:
                    result.append(n.val)

                if n.left:
                    stack.append(n.left)
                
                if n.right:
                    stack.append(n.right)

        return result


```

## 🔍 **程式碼解析**

### 外層 while

```python
whilelen(stack):


```

- `stack` 代表「目前這一層的所有節點」

---

### 內層 while（處理同一層）

```python
dq = deque(stack)
stack = []


```

- 用 `dq` 逐一 pop 目前層的節點
- 同時蒐集下一層的節點到 `stack`

---

### 關鍵判斷（右側節點）

```python
iflen(dq) ==0:
    result.append(n.val)


```

👉 因為 BFS 是 **由左到右** 處理

👉 當 `dq` 為空，代表 `n` 是該層最後一個（最右邊）

這個判斷 **邏輯完全正確** 👍

---

## ⏱ **時間與空間複雜度**

- **時間複雜度**：`O(n)`
    - 每個節點只拜訪一次
- **空間複雜度**：`O(n)`
    - queue / stack 在最寬層時可能存 n/2 個節點

---

## ✨ **更精煉的 BFS 寫法（Notion 可選）**

概念一樣，但用 `queue size` 來標示層級：

```python
from collections import deque

class Solution:
    def rightSideView(self, root):
        if not root:
            return []

        q = deque([root])
        res = []

        while q:
            size = len(q)

            for i in range(size):
                node = q.popleft()

                if i == size - 1:
                    res.append(node.val)

                if node.left:
                    q.append(node.left)
                if node.right:
                    q.append(node.right)

        return res


```

---

## 🔄 **DFS 解法（補充觀念）**

用 DFS（先走右邊）也可以：

```python
class Solution:
    def rightSideView(self, root):
        res = []

        def dfs(node, depth):
            if not node:
                return

            if depth == len(res):
                res.append(node.val)

            dfs(node.right, depth + 1)
            dfs(node.left, depth + 1)

        dfs(root, 0)
        return res


```

---

## ⚠️ **常見錯誤**

- 忘記只取每一層一個值
- DFS 沒控制 depth
- BFS 沒正確分層
- 把「右側視角」誤解成「只走右子樹」

---

## 📘 **一句話總結（Notion 摘要）**

> Binary Tree Right Side View =
每一層 BFS 的最後一個節點（或 DFS 先走右邊）。
