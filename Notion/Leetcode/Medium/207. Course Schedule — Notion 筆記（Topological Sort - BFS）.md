## ⭐ **難度：Medium (6/10)**

## 📌 **面試常考程度：9/10（圖論必考）**

只要題目出現以下關鍵字，幾乎一定是在考這題型：

- prerequisite / dependency
- course schedule / task order
- 能不能完成 / 是否有 cycle

---

## 🎯 **題意**

給定：

- `numCourses`：課程總數（0 ～ numCourses-1）
- `prerequisites`：`[course, prerequisite]`

判斷是否 **可以完成所有課程**。

---

## 💡 **核心觀念：Topological Sort（拓樸排序）**

### 一句話版本：

> 如果一張「先修關係圖」沒有 cycle（是 DAG），
就可以完成所有課程。

207 不要求你輸出順序，只問：

👉 **這張圖有沒有 cycle？**

---

## 🧠 **關鍵資料結構**

### 1️⃣ `graph`（Adjacency List）

```python
graph[pre] = [course1, course2, ...]


```

表示：

> 修完 pre 之後，可以修哪些課

---

### 2️⃣ `indegree`

```python
indegree[i] = 課程 i 還有幾個先修課沒完成


```

- `indegree == 0` → 可以立刻修
- `indegree > 0` → 還不能修

---

### 3️⃣ Queue（BFS）

- 存放所有 `indegree == 0` 的課程
- 代表「目前可以修的課」

---

## 🔄 **整體流程（Kahn’s Algorithm）**

1. 建立 `graph` 與 `indegree`
2. 將所有 `indegree == 0` 的課加入 queue
3. 不斷從 queue 取出課程：
    - 視為「修完」
    - 將其指向的課程 `indegree -= 1`
    - 若某課 `indegree == 0` → 加入 queue
4. 統計修過的課程數
5. 判斷是否修完全部課程

---

## 🧩 **你的程式碼（標準且正確）**

```python
from collectionsimport deque

classSolution:
defcanFinish(self, numCourses: int, prerequisites:List[List[int]]) ->bool:
        indegree = [0] * numCourses
        graph = [[]for _inrange(numCourses)]
        dq = deque()

# 建圖 + 計算 indegree
for course, prein prerequisites:
            graph[pre].append(course)
            indegree[course] +=1

# 將所有 indegree = 0 的課加入 queue
for ninrange(numCourses):
if indegree[n] ==0:
                dq.append(n)

        total =0

# BFS（Topological Sort）
while dq:
            course = dq.popleft()
            total +=1

for cin graph[course]:
                indegree[c] -=1
if indegree[c] ==0:
                    dq.append(c)

return total == numCourses


```

---

## 🔍 **逐段重點解析**

### ✔ 建圖 + indegree

```python
graph[pre].append(course)
indegree[course] +=1


```

代表：

```plain text
pre → course


```

---

### ✔ queue 初始化

```python
if indegree[n] ==0:
    dq.append(n)


```

表示：

> 這些課 沒有任何先修條件

---

### ✔ BFS 過程

```python
course = dq.popleft()
total +=1


```

代表：

> 修完一門課

```python
indegree[c] -=1


```

代表：

> 「你的一個先修課已完成」

---

### ✔ 最終判斷（關鍵）

```python
return total == numCourses


```

- ✔ 相等 → 沒有 cycle → 可以修完
- ❌ 不相等 → 有 cycle → 無法完成

---

## ⏱ **時間與空間複雜度**

- **時間複雜度**：`O(V + E)`
    - V = numCourses
    - E = len(prerequisites)
- **空間複雜度**：`O(V + E)`
    - graph + indegree + queue

---

## ❌ **什麼時候會失敗？（Cycle）**

如果：

- 所有剩餘課程 `indegree > 0`
- queue 提早變空
- `total < numCourses`

👉 代表圖中有 cycle

---

## 📝 **常見錯誤**

- 建圖方向反了
- 忘記初始化 indegree
- 沒用 indegree == 0 當起點
- 以為一定要真的輸出順序（207 不用）

---

## 📘 **一句話總結（Notion 摘要）**

> Course Schedule 的本質是：
用 Topological Sort 檢查課程依賴圖是否有 cycle。
能處理完所有 indegree 為 0 的節點，就能完成所有課程。
