## ⭐ **難度：Easy–Medium (4/10)**

## 📌 **面試常考程度：8/10**

這題是 **Two Pointers 的經典入門題**，

幾乎是「排序陣列 → 雙指標」的必考代表。

---

## 🎯 **題意**

給一個 **已排序（遞增）** 的整數陣列 `numbers`，

找出兩個數，使它們的和等於 `target`。

限制：

- **每個輸入只會有一組解**
- 回傳的是 **1-based index**
- 不能重複使用同一個元素

---

## 💡 **核心思路：Two Pointers（雙指標）**

因為陣列已排序，可以利用大小關係來移動指標：

- `left` 指向最左（最小）
- `right` 指向最右（最大）

每次比較：

```plain text
numbers[left] + numbers[right]


```

---

## 🔄 **指標移動規則**

- 若 `curr == target` → 找到答案
- 若 `curr < target` → 和太小，`left += 1`
- 若 `curr > target` → 和太大，`right -= 1`

👉 每一步都能 **排除一整段不可能的解**

---

## 🧠 **你的解法（標準且正確）**

```python
classSolution:
deftwoSum(self, numbers: List[int], target:int) ->List[int]:
        left =0
        right =len(numbers) -1

while left < right:
            curr = numbers[left] + numbers[right]

if curr == target:
return [left +1, right +1]

if curr < target:
                left +=1
else:
                right -=1


```

---

## 🔍 **程式碼解析**

- 使用 `left < right` 確保不重複使用同一元素
- 回傳時 `+1` 是因為題目要求 **1-based index**
- 題目保證一定有解，因此不需要額外 return fallback

---

## ⏱ **時間與空間複雜度**

- **時間複雜度**：`O(n)`
    - 每個指標最多走完整個陣列一次
- **空間複雜度**：`O(1)`
    - 只使用常數額外空間

---

## ⚠️ **常見錯誤**

- 忘記陣列是 **1-based index** 回傳
- 用 HashMap（可行但不是最佳解）
- 用巢狀 for loop（O(n²)，不必要）
