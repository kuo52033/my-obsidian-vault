**難度**：Medium

**題型**：Linked List

---

### 💡 題目說明

給定兩個**非空**的鏈結串列，代表兩個非負整數。

每個節點只存一位數字，數字是**反向儲存**的（個位數在最前面）。

請回傳相加後的結果（同樣以反向鏈結串列表示）。

---

### 🔍 範例

**輸入：**

```plain text
l1 = [2,4,3]
l2 = [5,6,4]
```

**輸出：**

```plain text
[7,0,8]
```

**解釋：**

342 + 465 = 807

---

### 🧠 解題思路

1. 使用指標同時遍歷兩條 linked list
2. 每次取出當前位數的值相加，再加上上一輪的 **carry（進位）**
3. 新節點的值為 `(v1 + v2 + carry) % 10`
4. 更新 carry 為 `(v1 + v2 + carry) // 10`
5. 直到兩條 list 都走完且 carry 為 0 才結束

---

### 🧩 運用到的技巧

- Linked List 的遍歷與指標操作
- Carry（進位）的維護
- Dummy head（虛擬頭節點）技巧：方便構造結果串列
- 模擬數學加法過程

---

### 🧾 程式碼

```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next

class Solution:
    def addTwoNumbers(self, l1: Optional[ListNode], l2: Optional[ListNode]) -> Optional[ListNode]:
        carry = 0
        result = ListNode()
        curr = result

        while l1 or l2 or carry:
            v1 = l1.val if l1 else 0
            v2 = l2.val if l2 else 0

            newNum = v1 + v2 + carry
            carry = newNum // 10
            curr.next = ListNode(newNum % 10)

            l1 = l1.next if l1 else None
            l2 = l2.next if l2 else None
            curr = curr.next

        return result.next


```

---

### ⏱️ 時間複雜度

**O(max(m, n))**

需遍歷兩個鏈結串列的每個節點一次

### 💾 空間複雜度

**O(max(m, n))**

輸出結果需要建立新的鏈結串列節點

---

### ✅ 重點複習

- Dummy head 可以避免處理第一個節點的特例
- 注意最後一位相加後仍可能有進位（需要額外節點）
- 不需轉成整數運算（因為數字可能超出整數範圍）
