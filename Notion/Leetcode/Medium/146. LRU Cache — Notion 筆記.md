---

# 🎯 **題意（重點）**

設計一個 LRU（Least Recently Used）快取：

- `get(key)` → 回傳 value，並標記為最近使用
- `put(key, value)` → 新增或更新 key
- 若容量滿，需刪除 **最久未使用（LRU）項目**

要求：

- `get()` 和 `put()` 都要 **O(1)** time complexity
- 不可使用額外 array 或排序（會超時）

---

# 💡 **最佳解法：HashMap + Doubly Linked List（雙向鏈表）**

---

## **核心結構**

### **1. HashMap (key → node)**

讓你能 O(1) 找到該節點。

### **2. Doubly Linked List（雙向 linked list）**

維護 LRU 順序：

```plain text
head(dummy) <-> newest ... oldest <-> tail(dummy)


```

- `head.next` = 最新使用 (MRU)
- `tail.prev` = 最久未使用 (LRU)

這樣可以：

- O(1) 插入節點到最前面（MRU）
- O(1) 移除任意節點（因為雙向鏈結）
- O(1) 刪除尾端的 LRU 節點

---

# 🧩 **三種核心操作**

### **1️⃣ move to head**

將某節點標記為最近使用。

### **2️⃣ remove from list**

從 linked list 中 O(1) 移除任意節點。

### **3️⃣ remove LRU when full**

`tail.prev` 就是 LRU，刪除即可。

# 🧠 **你的完整正確解（最佳版本）**

```python
class Node:
    def __init__(self, key, val):
        self.val = val
        self.key = key
        self.prev = None
        self.next = None

class LRUCache:

    def __init__(self, capacity: int):
        tail = Node(-1, -1)
        head = Node(-1, -1)
        tail.prev = head
        head.next = tail

        self.hashTable = {}
        self.capacity = capacity
        self.tail = tail    # dummy tail
        self.head = head    # dummy head

    def get(self, key: int) -> int:
        if key not in self.hashTable:
            return -1

        node = self.hashTable[key]
        self.deleteNode(node)
        self.addNode(node)
        return node.val

    def put(self, key: int, value: int) -> None:
        if key in self.hashTable:
            node = self.hashTable[key]
            node.val = value
            self.deleteNode(node)
            self.addNode(node)
            return

        # evict LRU
        if len(self.hashTable) == self.capacity:
            lru = self.tail.prev
            self.deleteNode(lru)
            del self.hashTable[lru.key]

        newNode = Node(key, value)
        self.addNode(newNode)
        self.hashTable[key] = newNode

    def deleteNode(self, node):
        node.prev.next = node.next
        node.next.prev = node.prev

    def addNode(self, node):
        node.prev = self.head
        node.next = self.head.next
        self.head.next.prev = node
        self.head.next = node


```

# ⏱ **時間複雜度**

| 操作 | 複雜度 |
| --- | --- |
| get | O(1) |
| put | O(1) |
| 新增節點 | O(1) |
| 刪除節點 | O(1) |

總結：

### 👉 **所有操作都是 O(1)**（這也是本題的核心）
