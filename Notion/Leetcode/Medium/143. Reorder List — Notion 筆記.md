## ⭐ **難度：Medium (6/10)**

## 📌 **面試常考程度：8/10**

這題是 Linked List 三大經典題之一（另兩題：143、148），

考到的技巧非常基本但重要：

- 快慢指標找中點
- 反轉鏈表
- 合併兩條鏈表
- in-place 修改（不能用 array 複製）

---

# 🎯 **題意**

給定鏈表：

```plain text
L0 → L1 → L2 → ... → Ln
```

要重新排列成：

```plain text
L0 → Ln → L1 → Ln-1 → L2 → Ln-2 → ...
```

不開新 list，只能靠指標操作。

---

# 💡 **解題總覽（經典三步）**

---

## **1️⃣ 找中點 — fast/slow pointer**

使用 fast、slow 走法：

```plain text
slow = head
fast = head

while fast and fast.next:
    slow = slow.next
    fast = fast.next.next


```

`slow` 最後停在中間位置。

---

## **2️⃣ 反轉後半段**

從 slow 後面開始反轉，將後半變為：

```plain text
倒序鏈表 reversed_second_half
```

---

## **3️⃣ 交錯合併兩段**

前半段：

```plain text
1 → 2 → 3
```

反轉後半段：

```plain text
5 → 4
```

合併成：

```plain text
1 → 5 → 2 → 4 → 3


```

小心指標不要被覆蓋導致循環或斷開。

---

# 🧠 **你的程式碼（逐段解析）**

```python
class Solution:
    def reorderList(self, head: Optional[ListNode]) -> None:
        if not head:
            return


```

### ✔ Check 空 list

---

## **Part 1：找中點**

```python
        fast = head
        slow = head

        while fast and fast.next:
            slow = slow.next
            fast = fast.next.next


```

`slow` 會停在中間點（前半段的尾）。

---

## **Part 2：反轉後半段**

```python
        curr = slow.next
        prev = None

        while curr:
            temp = curr.next
            curr.next = prev
            prev = curr
            curr = temp

        slow.next = None


```

反轉 slow 後面的所有節點 → `prev` 是後半段的反轉頭。

並把 `slow.next = None` 斷開兩段鏈表。

---

## **Part 3：合併前半與後半（交錯）**

```python
        while head and prev:
            temp1 = head.next
            temp2 = prev.next

            head.next = prev
            prev.next = temp1

            head = temp1
            prev = temp2


```

這段是整題的核心：

交錯合併，兩個 pointer 各自前進。

程式碼完全正確，沒有 edge case 問題。

---

# ⏱ **時間複雜度**

### **O(n)**

- 找中點 O(n)
- 反轉鏈表 O(n)
- 合併 O(n)

---

# 📦 **空間複雜度**

### **O(1)**

完全 in-place。
