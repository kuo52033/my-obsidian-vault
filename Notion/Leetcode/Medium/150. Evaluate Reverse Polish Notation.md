# 🎯 **題意（重點）**

給定一組逆波蘭表示式（Reverse Polish Notation, RPN），

請計算最後的結果。

例：

```plain text
["2","1","+","3","*"] → (2 + 1) * 3 = 9

```

運算子永遠接在後面，運算順序由 stack 處理。

---

# 💡 **解題策略（核心概念）**

使用 **stack**：

1. 遇到數字 → push 進 stack
2. 遇到運算子 → pop 兩個數，計算後 push 回去
3. 最後 stack 裡剩一個值 → return

注意：

- 遇到 "-" 或 "/" 時，順序很重要
因為 RPN 是：

```plain text
v1 (先彈出) operator v2 (後彈出)
```

---

# ⚠️ **Python 除法的陷阱（本題最大重點）**

RPN 題目要求：

> 除法要「向零截斷」(truncate toward zero)

Python 的 `/` 是浮點除法

Python 的 `//` 是 floor 除法（對負數會不同結果）

例如：

```plain text
-3 // 2 = -2   (Floor)
int(-3 / 2) = -1   (Truncate toward zero)
```

### ✔ 本題必須用 `int(v1 / v2)`

你的解法使用這個技巧 → 完全正確。
