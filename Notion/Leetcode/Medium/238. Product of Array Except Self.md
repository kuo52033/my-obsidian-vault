## 📌 題目重點

- 回傳一個陣列 `result`
- `result[i]` = 陣列中 **除了 **`**nums[i]**`** 以外的所有元素乘積**
- **不可使用除法**
- 要求 **O(n) 時間**
- 進階要求：**O(1) 額外空間（不含輸出）**

---

## 🧠 解題核心思路（Prefix × Suffix）

對每個 index `i`：

```plain text
result[i]= (左邊所有元素的乘積) × (右邊所有元素的乘積)


```

---

## 🧩 使用到的資料結構

- **Array（result）**
- **一個變數（right）**
- 沒有額外 prefix / suffix 陣列（空間最佳化）

---

## 🧪 程式碼解析（逐段）

### 🔹 初始化

```python
result = [1] *len(nums)


```

- 先把結果陣列設成全 1
- 方便後續直接用 `=` 累乘

---

### 🔹 第一輪：Prefix Product（左邊）

```python
for iinrange(1,len(nums)):
    result[i] *= result[i-1] * nums[i-1]


```

此時：

- `result[i]` = `nums[0] * nums[1] * ... * nums[i-1]`
- `result[0] = 1`（左邊沒有元素）

✅ 邏輯完全正確

---

### 🔹 第二輪：Suffix Product（右邊）

```python
right =1
for iinrange(len(nums)-1, -1, -1):
    result[i] *= right
    right *= nums[i]


```

- `right` 代表「i 右邊所有元素的乘積」
- 每走一步：
    - 先用目前的 `right`
    - 再把 `nums[i]` 乘進去，供下一輪使用

✅ 更新順序正確（這點很多人會寫錯）

---

### 🔹 回傳結果

```python
return result


```

---

## 🧠 用範例快速驗證

`nums = [1, 2, 3, 4]`

### Prefix 後

```plain text
result = [1,1,2,6]


```

### Suffix 掃描

```plain text
right =1
i=3 → result[3]=6*1=6, right=4
i=2 → result[2]=2*4=8, right=12
i=1 → result[1]=1*12=12, right=24
i=0 → result[0]=1*24=24


```

結果：

```plain text
[24, 12, 8, 6]


```

---

## ⏱️ 時間複雜度

- 兩次 linear scan

✅ **Time Complexity：O(n)**

---

## 💾 空間複雜度

- 除了輸出陣列，只使用一個變數 `right`

✅ **Space Complexity：O(1)**（不含 result）

---

## 🎯 面試加分點（你已經做到）

- ❌ 不用除法
- ✅ 處理 0 的情況自然正確
- ✅ O(n) time
- ✅ O(1) extra space
- ✅ 清楚的 prefix / suffix 拆解

---

## 🔄 常見錯誤（你完全避開）

- 右邊乘積更新順序寫反
- 多開一個 `right[]` 陣列
- 忘記 `result[0] = 1`
- 嘗試用除法處理 0
