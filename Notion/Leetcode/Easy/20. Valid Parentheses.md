- **核心標籤**: `堆疊 (Stack)`, `字串 (String)`
- **題目精髓**: 處理具有「對稱性」或「後進先出 (LIFO)」特性的匹配問題。
- **關鍵思想：利用堆疊的 LIFO 特性**
    - 括號問題的本質是，**最晚出現的左括號，必須被最早出現的右括號匹配**。
    - 這個特性與堆疊 (Stack) 的 `append` (push) 和 `pop` 操作完美契合。
- **演算法模式 (Algorithm Pattern)**:
    1. 建立一個堆疊 `stack` (在 Python 中用 `list` 即可模擬)。
    2. 遍歷輸入的字串 `s`。
    3. **遇到左括號**: 將其**壓入 (push)** 堆疊中。`stack.append(char)`
    4. **遇到右括號**:
        - **檢查堆疊是否為空**: 如果為空，代表沒有匹配的左括號，直接判斷為無效。
        - **彈出 (pop)** 堆疊頂部的元素，與當前的右括號進行比對。
        - 如果不匹配，判斷為無效。
    5. **遍歷結束後**: **檢查堆疊是否為空**。如果為空，代表所有括號都完美匹配；如果不為空，代表有未閉合的左括號，判斷為無效。
- **Python 實用技巧**:
    - **用 **`**list**`** 模擬堆疊**: `append()` 即為 `push`，`pop()` 即為 `pop`。
    - **雜湊表輔助匹配**: 建立一個 `mapping = {')':'(', ']':'[', '}':'{'}`，可以 O(1) 地找到右括號對應的左括號。
    - **真值測試 (Truthiness)**: `if stack:` 用來判斷堆疊是否為空；`return not stack` 是判斷最終結果的簡潔寫法。
    - **三元運算子**: `stack.pop() if stack else '#'` 可以優雅地處理「嘗試從空堆疊 pop」的邊界情況，避免程式出錯。
- * takeaway**:
> 凡是遇到需要「配對」、「消除」或具有「最近相關性」的問題（如括號匹配、XML/HTML 解析、計算機的撤銷操作），都應該立刻聯想到「堆疊」。
