畢業於輔仁大學資工系，在校期間參與推薦系統專題開發，負責後端資料爬蟲、機器學習模型訓練與 Monte Carlo Tree Search（MCTS）演算法實作，以遊戲對戰數據為基礎，訓練推薦模型分析陣容搭配，協助玩家提升勝率。這也奠定了對系統後端與資料應用的興趣。
畢業後進入軟體公司擔任後端工程師，前一年半主要為企業客戶開發客製化功能，使用 Node.js、WebSocket、MySQL、MongoDB 等技術，並負責第三方 API 串接，將客戶需求轉化為穩定運行的產品功能。這段經歷讓我建立了後端開發的技術基礎，以及與 PM、前端工程師協作交付產品的實務能力。
